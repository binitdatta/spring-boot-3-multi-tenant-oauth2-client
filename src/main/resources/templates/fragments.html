    IAM

    Identity and Access Management. Identity is the Server checking who we are. Access is whether we are allowed to do something and Management is doing Identity Checking Access from Thousands of users.

    OAuth2

    Open Authorization Protocol Version 2. OAuth2 is delegating all Authorization Decisions to a Central Identity Provider. If you saw websites that allows you to login using Google/LinkedIn, You have seen OAuth2

    Spring Boot 3

    Has made JDK 17 mandatory and changed the package of jpa and jee from javax to jakarta. It also provides huge boost to WebClient which is both Synchronous and Asynchronous.

    Spring Security 6

    From Spring Security 5, the OAuth2 Dependencies are now separate. We now can have a thin Resource Server (aka API) and a OAuth2 Client

    Security Config

    HAs changed a bit from Spring Security 5. We are now required to configure and return a SecurityFilterChain. Details on the Menu.

    KeyCloak 21.1.1

    KeyCloak is an Open Source IAM Server from IBM and Red Hat. KeyCloak is the Microsoft Office , an undisputed leader in the IAM World. Free and Feature Rich. Details on the Menu.
