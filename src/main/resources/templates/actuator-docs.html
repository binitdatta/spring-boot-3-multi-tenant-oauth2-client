Spring Boot (3) Actoator

  • Spring Boot comes with its own monitoring system called Actuator
  • if we have included a dependency named spring-boot-starter-actuator and include a few properties, it will work
  • The default Actuator endpoint for this service is : http://localhost:19090/actuator
  • To see all Beans that Spring Configured for us Hit : http://localhost:19090/actuator/beans
  • If there is a local cache manager, we can see that in : http://localhost:19090/actuator/caches
  • To check the default health of the Service, hit : http://localhost:19090/actuator/health
  • To check if the Service is up or running or get some information, hit : http://localhost:19090/actuator/info
  • To see all conditional beans : Hit : http://localhost:19090/actuator/conditions
  • To see Log Levels , Hit : http://localhost:19090/actuator/loggers
  • To get a heapsump, hit : http://localhost:19090/actuator/heapdump
  • To get a threaddump, hit : http://localhost:19090/actuator/threaddump
  • To get the names all metrics : http://localhost:19090/actuator/metrics
  • To get a metric of application ready time , hit : http://localhost:19090/actuator/metrics/application.ready.time
  • To get a metric of application started time , hit : http://localhost:19090/actuator/metrics/application.started.time
